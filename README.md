# Portfolio Architetto - Astro (TypeScript)

Scheletro iniziale di un sito Astro statico, senza design e senza contenuti.

## Requisiti

- Node.js 18+ (consigliato LTS recente)
- npm

## Avvio in locale

```bash
npm install
npm run dev
```

Il server di sviluppo parte su `http://localhost:4321`.

## Build di produzione

```bash
npm run build
```

L'output statico viene generato nella cartella `dist/`.
